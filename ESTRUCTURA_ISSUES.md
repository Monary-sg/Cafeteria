## Flujo de Trabajo

1. **CREAR** → Issue en estado "open"
2. **DISEÑAR** → Agregar label "diseño" + descripción
3. **DESARROLLAR** → Label "backend" o "frontend" + estimación
4. **PROBAR** → Label "testing"
5. **COMPLETAR** → Label "completado" + cerrar issue
